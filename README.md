# Calculator

IMPORTANT: If you want to run the calculator website you have to download and unzip the three files (calculator.js, index.html, style.css). Opening only the index.html inside the zipped file will open the page but without CSS, which is quite ugly.

About this repository: I made the code logic and lines in C, classical C. However, I wanted to create an interactive fancy version with a front end on a website. So, I showed the code to Claude AI and it produced the front end of the system as well as translating the code to JavaScript. I was a little disappointed my beloved C is not enough, but anyways...
